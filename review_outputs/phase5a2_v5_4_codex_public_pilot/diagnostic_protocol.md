## Blindness

The same fresh Generator that stopped before receiving any WorkPacket may be
continued because it has seen only the unchanged CASE and installed skill. It
must not receive V5.3 failure facts, this repository's evaluation key, prior
candidates or audits, source code, tests, web material, or case-specific
economic hints. It may not use subagents.

The Generator is told only that this is the separately labelled V5.4-D1
diagnostic, that exact model-name observation is waived, and that it must apply
the unchanged CASE and skill in the new clean root. This message is part of the
diagnostic treatment and prevents the result from being mislabelled as the
official run.
